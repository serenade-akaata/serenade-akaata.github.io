HTML, CSS and The DOM

What Is an analogy for describing the differences between HTML and CSS:

HTML and CSS are used for website presentation, so think of HTML as the structure of your webpage (content) and CSS as the styles (font, colour, etc). When trying to understand what the differences of HTML and css are, try thinking of it as building a house. You have the frame, walls, roof, bedrooms, entrance, kitchen, etc., this would be your HTML. Whereas the styling, such as decorations, how tall walls are and the colour of them, would be the CSS. So what I’m trying to say is HTML and CSS are quite like building a house. When it comes to interacting with things in your house, like turning on lights and starting the dishwasher, this would be JavaScript.

Control flow and looping

A control flow in programs are just the order which the programs codes executes. So basically having the first action and going down to the last execution, until the action or task is complete. This is where loops come in. Loops is a function which executes a repetitive task. Here’s an example:

Going for a walk:

  1. Putting shoes on
  2. Move left foot forward
  3. Move right foot forward
  4. Move left foot forward
  5. Move right foot forward
  6. Move left foot forward
  7. Move right foot forward
  8. Move left foot forward
  9. Move right foot forward
  10. Move left foot forward
  11. Move right foot forward
  12. Reached destination

Instead of writing this out, you can do these steps in-between repeatedly via looping, until you’ve reached your destination.

What is the DOM

Document Object Model provides the means to interact with objects in an HTML document. Everything that makes up your DOM is more generically known as nodes. These nodes can be elements, attributes, text, content, comments, document-related stuff, and various other things you simply never think about.

Ever heard of the term “tree-structure”? If not you’ll hear it a lot in working with the DOM. The DOM organises your html into a tree structure, which each element then becomes a separate node and the way to access this would be to open your developer tools, which is built into browsers at the top right of any website you’re on. You can use this feature to manipulate and change elements of your choice.

Arrays and Objects

Firstly what is an array? Arrays are like a collection of a fixed number of values of a single type. The way to access the data within an array is typing the array name and the position you want the data form. Example:

var arrayname = [item 1, item 2, item 3]
arrayname[0]

This will display the arrayname and item 1, because computers start with the value 0 instead 1. So item 1 is really number 0.
